Triagem Inteligente para Recrutadores

    Você sabia que uma empresa gasta em média 24 horas e mais de R$ 1.000,00 para apenas filtrar um novo talento em uma única vaga? Pensando nisso, resolvi criar um web app com o objetivo de integrar uma inteligência artificial capaz de fazer a triagem inteligente de pessoas em processos seletivos, ou seja, aplica uma filtragem e também seleciona os candidatos mais semelhantes ao que o empregador busca, possibilitando a reduzir o tempo do processo para segundos e o custos em até 50%.

    É importante lembrar que esse web app é uma demonstração de um trabalho pessoal, mas que pode se tornar um serviço muito útil no dia a dia de empresas com grande número de vagas a oferecer.


    Como utilizar:

    - Cadastre um talento (se não quiser, o algoritmo utiliza de uma base de dados já criada);

    - Cadastre uma vaga;

    - Partindo das informações da vaga que você cadastrou, o algoritmo utiliza de Processamento de Linguagem Natural (NLP) para filtrar os melhores candidatos.
